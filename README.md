# 人生网格 (LifeGrid)

一个诗意的人生旅程可视化工具。记录你的人生足迹，设定未来的愿望，并向未来的自己发送一封跨越时空的信件。

## 功能

- **人生旅程可视化**：以月和年为单位查看你的人生。
- **寄往未来的信**：写给未来自己的信，在特定年龄解锁。
- **人生目标**：设定并追踪你的人生里程碑。
- **时光证书**：为你的时间胶囊生成精美的证书。
- **实时同步**：使用 Supabase 在不同设备间同步你的数据。

## 技术栈

- React 19
- Vite
- Tailwind CSS 4
- Supabase (Auth & Database)
- Motion (Framer Motion)
- Lucide Icons

## 开始使用

1. 克隆仓库。
2. 安装依赖：`npm install`。
3. 在 AI Studio 设置中配置 `VITE_SUPABASE_URL` 和 `VITE_SUPABASE_ANON_KEY`。
4. 运行开发服务器：`npm run dev`。
